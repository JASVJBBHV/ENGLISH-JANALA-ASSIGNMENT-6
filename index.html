1. Write the difference between var, let, and const.

we can use var outside the function even if it is declared within the function it won't show any error .it is function scoped and can be redeclared. on the other hand let can be updated but not redeclared. while const is block scoped. we can not use it or update it outside the function

map() creates a new array by doing operation to each element and returns the modified values. foreach() executes a function on each element but does not return anything like we can loop through each element of given input but can not do anything with them. filter() creates a nww array containing only the elements that meets the condition. arrao function is lexically blind. which means they inherit it from sorrounding context.
